# MuffinStore
带有TrollStore的实验性App Store应用降级/安装程序

要下载旧版本，您可以选择手动输入应用旧版本的标识符，或从API获取它们。您可能会考虑从API获取它们会侵犯隐私，因为它还需要发送您想要与请求一起下载的应用程序ID。如果您想在不使用App Store以外的外部API的情况下使用此应用程序，您只需手动输入ID即可。你可以从ipatool-py和其他地方获得它们
